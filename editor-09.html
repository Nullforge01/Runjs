Task: Create function getAdultNames that:

  1. Takes array of user objects: [{name: "Alex", age: 20},...]
  2. Filters users with age >= 18
  3. Maps to array of names only
  4. Returns that array

Example: getAdultNames([{name:"Ann",age:17},{name:"Bob",age:20}]) → ["Bob"]
