Lesson 6: Objects

easy · 40 XP

1. What Are Objects?

Objects group related data into key-value pairs. Arrays use numbered indexes. Objects use named keys.

// Object literal
const user = {
  name: "Alex",
  age: 25,
  isAdmin: true
};

2. Accessing Properties

Dot Notation — Use when you know the key

console.log(user.name); // "Alex"
user.age = 26; // Update

Bracket Notation — Use for dynamic keys

const key = "name";
console.log(user[key]); // "Alex"
user["isAdmin"] = false;

3. Adding & Deleting Properties

const car = { brand: "Tesla" };
car.model = "Model 3"; // Add new key
car.year = 2024;

delete car.year; // Remove key
console.log(car); // { brand: "Tesla", model: "Model 3" }

4. Methods — Functions Inside Objects

When a function is a property, it’s called a method. Use this to access other properties.

const person = {
  name: "Sarah",
  greet: function() {
    return `Hi, I'm ${this.name}`;
  },
  // Shorthand method syntax
  sayAge() {
    return `I'm 30`;
  }
};

console.log(person.greet()); // "Hi, I'm Sarah"

5. Nested Objects & Arrays

const profile = {
  username: "dev_pro",
  stats: {
    posts: 42,
    followers: 1200
  },
  skills: ["JS", "React", "Node"]
};

console.log(profile.stats.posts); // 42
console.log(profile.skills[1]); // "React"

6. Looping Through Objects

const scores = { math: 90, english: 85, science: 95 };

// for...in — loop keys
for (const subject in scores) {
  console.log(`${subject}: ${scores[subject]}`);
}

// Object.keys — get array of keys
const keys = Object.keys(scores); // ["math", "english", "science"]
// Object.values — get array of values
const values = Object.values(scores); // [90, 85, 95]

7. Common Mistakes

  • Undefined property: user.email returns undefined if key doesn’t exist
  • Mutating const objects: const obj = {} lets you change obj.x = 1. You just can’t reassign obj = {}
  • Dot vs bracket: obj.123 fails. Use obj["123"] for invalid identifier keys

8. Real Examples

// Product object
const product = {
  id: 101,
  name: "Laptop",
  price: 999,
  inStock: true,
  tags: ["tech", "work"],
  getDiscount: function(percent) {
    return this.price * (1 - percent / 100);
  }
};

console.log(product.getDiscount(10)); // 899.1

// Config object
const settings = {
  theme: "dark",
  notifications: true,
  language: "en"
};
settings.theme = "light"; // Update setting

Mastery Checklist:

  • Create objects with key-value pairs
  • Access/update with dot and bracket notation
  • Add and delete properties
  • Write methods using this
  • Loop objects with for...in and Object.keys
