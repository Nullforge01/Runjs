Lesson 3: Conditionals

easy · 30 XP

1. Making Decisions with Code

Conditionals let your code choose different paths based on true/false checks. Like "if it's raining, take umbrella".

const isRaining = true;

if (isRaining) {
  console.log("Take umbrella");
} else {
  console.log("Wear sunglasses");
}

2. if, else if, else

const score = 85;

if (score >= 90) {
  console.log("A grade");
} else if (score >= 80) {
  console.log("B grade"); // This runs
} else if (score >= 70) {
  console.log("C grade");
} else {
  console.log("F grade");
}

Rule: JS checks top to bottom. Stops at first true condition.

3. Comparison Operators

OperatorMeaningExample
===Equal value + type5 === 5 → true
!==Not equal5 !== 3 → true
>Greater than10 > 5 → true
<Less than3 < 7 → true
>=Greater or equal5 >= 5 → true
<=Less or equal4 <= 5 → true

Avoid == — it does type coercion. 5 == "5" is true but dangerous. Always use ===.

4. Logical Operators

AND && — Both must be true

const age = 20;
const hasID = true;

if (age >= 18 && hasID) {
  console.log("Can enter club"); // Runs
}

OR || — At least one true

const isWeekend = false;
const isHoliday = true;

if (isWeekend || isHoliday) {
  console.log("No work today"); // Runs
}

NOT ! — Flip boolean

const isBanned = false;

if (!isBanned) {
  console.log("User allowed"); // Runs
}

5. Ternary Operator — Short if/else

const age = 20;
const status = age >= 18 ? "Adult" : "Minor";
// condition ? valueIfTrue : valueIfFalse

console.log(status); // "Adult"

6. switch — Multiple exact matches

const day = "Monday";

switch (day) {
  case "Monday":
    console.log("Start of week");
    break;
  case "Friday":
    console.log("Weekend soon");
    break;
  default:
    console.log("Regular day");
}

Don’t forget break or it falls through to next case.

7. Truthy & Falsy

JS treats these as false: false, 0, "", null, undefined, NaN. Everything else is truthy.

const username = "";

if (username) {
  console.log("Has name");
} else {
  console.log("Empty"); // This runs
}

8. Real Examples

// Login check
function checkLogin(user, pass) {
  if (!user || !pass) {
    return "Missing fields";
  } else if (user === "admin" && pass === "1234") {
    return "Welcome admin";
  } else {
    return "Invalid credentials";
  }
}

// Discount logic
const total = 150;
const discount = total > 100 ? 0.1 : 0;
const finalPrice = total * (1 - discount);

Mastery Checklist:

  • Write if / else if / else chains
  • Use comparison ===, !==, >, < correctly
  • Combine conditions with && and ||
  • Use ternary for simple cases
  • Know when to use switch vs if
