Lesson 5: Arrays

easy · 35 XP

1. What Are Arrays?

Arrays store multiple values in a single variable. Like a list. Order matters. Zero-indexed.

const fruits = ["apple", "banana", "cherry"];
const numbers = [1, 2, 3, 4, 5];
const mixed = ["text", 42, true, null];

2. Accessing Elements

Index starts at 0. First item = 0, second = 1, etc.

const colors = ["red", "green", "blue"];

console.log(colors[0]); // "red"
console.log(colors[2]); // "blue"
console.log(colors[3]); // undefined - doesn't exist

colors[1] = "yellow"; // Change value
console.log(colors); // ["red", "yellow", "blue"]

3. Array Length

const items = ["a", "b", "c"];
console.log(items.length); // 3

// Get last item
console.log(items[items.length - 1]); // "c"

4. Core Array Methods

Add/Remove From End

const stack = [1, 2];
stack.push(3); // Add to end: [1, 2, 3]
const last = stack.pop(); // Remove from end: last = 3, stack = [1, 2]

Add/Remove From Start

const queue = [2, 3];
queue.unshift(1); // Add to start: [1, 2, 3]
const first = queue.shift(); // Remove from start: first = 1, queue = [2, 3]

Find Index/Check Exists

const nums = [10, 20, 30];
nums.indexOf(20); // 1 - position
nums.includes(40); // false - doesn't exist

Slice & Splice

const arr = [1, 2, 3, 4, 5];
arr.slice(1, 3); // [2, 3] - copy portion, original unchanged
arr.splice(2, 1); // Removes 1 item at index 2. arr = [1, 2, 4, 5]

5. Looping Through Arrays

const scores = [85, 90, 78];

// for loop - with index
for (let i = 0; i < scores.length; i++) {
  console.log(`Score ${i}: ${scores[i]}`);
}

// for...of - just values
for (const score of scores) {
  console.log(score);
}

// forEach - functional style
scores.forEach((score, i) => {
  console.log(`Index ${i}: ${score}`);
});

6. Common Mistakes

  • Off-by-one: arr[arr.length] is undefined. Last index is length - 1
  • Mutating vs copy: push/pop/splice change original. slice returns copy
  • const arrays: const arr = [1] lets you arr.push(2). You can't reassign arr = []

7. Real Examples

// Todo list
const todos = [];
todos.push("Learn JS"); // ["Learn JS"]
todos.push("Build app"); // ["Learn JS", "Build app"]
todos.shift(); // ["Build app"]

// Filter numbers > 10
const nums = [5, 12, 8, 20];
const big = [];
for (const n of nums) {
  if (n > 10) big.push(n);
}
console.log(big); // [12, 20]
