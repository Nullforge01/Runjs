Lesson 7: DOM Manipulation

medium · 45 XP

1. What is the DOM?

DOM = Document Object Model. The browser turns your HTML into a tree of objects that JS can read and change. Every tag is a node you can grab and edit.

<h1 id="title">Hello</h1>
// Becomes an object you can access with JS

2. Selecting Elements

// By ID - fastest, returns 1 element
const title = document.getElementById("title");

// By CSS selector - flexible, returns first match
const btn = document.querySelector(".btn-primary");

// By CSS selector - returns all matches as NodeList
const items = document.querySelectorAll("li");

Rule: Use getElementById for single IDs. Use querySelector for everything else.

3. Changing Content

const heading = document.getElementById("title");

// Text only - safe, no HTML parsing
heading.textContent = "New Title";

// HTML - renders tags, but risky with user input
heading.innerHTML = "New <span>Title</span>";

Security: Never set innerHTML with user input. XSS risk. Use textContent.

4. Changing Attributes & Styles

const img = document.querySelector("img");

// Attributes
img.src = "/new-logo.png";
img.setAttribute("alt", "Logo");
img.getAttribute("src"); // "/new-logo.png"

// Classes
const box = document.getElementById("box");
box.classList.add("active");
box.classList.remove("hidden");
box.classList.toggle("dark"); // add if missing, remove if present

// Inline styles
box.style.backgroundColor = "red";
box.style.fontSize = "20px";

5. Creating & Removing Elements

// Create
const newDiv = document.createElement("div");
newDiv.textContent = "I'm new";
newDiv.classList.add("card");

// Add to page
document.body.appendChild(newDiv); // add to end
const parent = document.getElementById("container");
parent.prepend(newDiv); // add to start

// Remove
newDiv.remove(); // modern way
// old way: parent.removeChild(newDiv);

6. Events — Making Pages Interactive

const button = document.getElementById("saveBtn");

button.addEventListener("click", function() {
  console.log("Button clicked!");
});

// Arrow function version
button.addEventListener("click", () => {
  alert("Saved!");
});

// Common events: click, input, submit, mouseover, keydown

7. Reading Form Input

<input id="username" type="text">
<button id="submit">Go</button>

<script>
const input = document.getElementById("username");
const btn = document.getElementById("submit");

btn.addEventListener("click", () => {
  const name = input.value; // .value for inputs
  console.log(`Hello ${name}`);
});
</script>

8. Common Mistakes

  • Null error: document.getElementById("oops").textContent crashes if ID doesn't exist. Check first.
  • Script runs too early: Put <script> before </body> or use defer
  • NodeList vs Array: querySelectorAll returns NodeList. Use forEach or spread [...items]

9. Real Example: Todo Add

<input id="task">
<button id="add">Add</button>
<ul id="list"></ul>

<script>
const input = document.getElementById("task");
const btn = document.getElementById("add");
const list = document.getElementById("list");

btn.addEventListener("click", () => {
  if (input.value.trim() === "") return;
  
  const li = document.createElement("li");
  li.textContent = input.value;
  list.appendChild(li);
  input.value = ""; // clear input
});
</script>

  • Add/remove classes with classList
