Lesson 11: LocalStorage & State

medium · 50 XP

1. What is State?

State = data your app remembers. Refresh the page and variables reset. LocalStorage keeps data after reload.

// Variable - gone on refresh
let count = 0;

// LocalStorage - survives refresh
localStorage.setItem("count", 0);

2. LocalStorage Basics

Key-value store in the browser. Only stores strings. ~5MB limit. Synchronous.

// Save
localStorage.setItem("username", "Alex");
localStorage.setItem("age", "25"); // converts to string

// Read
const name = localStorage.getItem("username"); // "Alex"
const age = localStorage.getItem("age"); // "25" - string!

// Remove
localStorage.removeItem("age");

// Clear all
localStorage.clear();

3. Storing Objects & Arrays

Must convert to JSON string. Parse when reading back.

const user = { name: "Alex", score: 100 };

// Save: stringify
localStorage.setItem("user", JSON.stringify(user));

// Read: parse
const saved = localStorage.getItem("user");
const userObj = saved ? JSON.parse(saved) : null;
console.log(userObj.name); // "Alex"

4. Pattern: Load or Default

Always check if data exists before using. Provide fallback.

// Get todos or empty array
const todos = JSON.parse(localStorage.getItem("todos") || "[]");

// Get settings or defaults
const theme = localStorage.getItem("theme") || "dark";

5. Pattern: Save on Change

Update localStorage every time state changes.

let todos = JSON.parse(localStorage.getItem("todos") || "[]");

function addTodo(text) {
  todos.push({ text, done: false });
  localStorage.setItem("todos", JSON.stringify(todos));
  render();
}

7. Common Mistakes

  • Forgetting JSON: localStorage.setItem("data", obj) saves "[object Object]". Use JSON.stringify
  • Type coercion: localStorage.getItem("count") returns string. Use parseInt() for numbers
  • Null checks: JSON.parse(localStorage.getItem("x")) crashes if "x" doesn't exist. Check first
  • Quota exceeded: 5MB limit. Don't store huge data

8. Real Example: Dark Mode Toggle

const btn = document.getElementById("themeBtn");
const body = document.body;

// Load saved theme on start
const savedTheme = localStorage.getItem("theme") || "light";
body.classList.add(savedTheme);

btn.addEventListener("click", () => {
  const isDark = body.classList.toggle("dark");
  body.classList.toggle("light");
  localStorage.setItem("theme", isDark ? "dark" : "light");
});

Mastery Checklist:

  • Save and read strings with setItem/getItem
  • Store objects with JSON.stringify/parse
  • Handle missing data with fallbacks
  • Sync app state to localStorage on change
  • Understand localStorage vs sessionStorage
