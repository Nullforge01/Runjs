What's Next: 3 Paths

1. Frontend Path: React

You already know 80% of React. It's just components + state + props. Your todo app becomes:

function TodoApp() {
  const [todos, setTodos] = useState([]);
  // same logic, JSX instead of DOM
}

Learn: React docs, Next.js, Tailwind CSS

2. Backend Path: Node.js

Same JS, but on servers. Replace localStorage with database.

app.post('/todos', (req, res) => {
  db.todos.push(req.body);
  res.json({ success: true });
});

Learn: Node.js, Express, PostgreSQL, REST APIs

3. Full-Stack Path: Build Projects

Best way to learn is shipping. Ideas:

  • Weather app: fetch API + localStorage for last city
  • Habit tracker: Dates, streaks, charts
  • Markdown notes: localStorage + live preview
  • Expense tracker: reduce for totals, filter by month

Key Habits of Good Devs

  1. Read error messages: They tell you exactly what's wrong
  2. Console.log everything: When confused, log the data
  3. Small steps: Make one thing work, then next thing
  4. MDN is your friend: Search "MDN array map" not "how to map"
  5. Build, don't just watch: Tutorials without coding = forgetting
