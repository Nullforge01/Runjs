Lesson 2: Functions

easy · 30 XP

1. What Are Functions?

Functions are reusable blocks of code that perform a specific task. Write once, run many times. Like a recipe you can follow whenever you need that dish.

// Function definition
function greet() {
  console.log("Hello!");
}

// Call the function
greet(); // prints: Hello!

2. Function Parts

function add(a, b) {  // a, b are parameters
  return a + b;       // return sends value back
}

const sum = add(5, 3); // 5, 3 are arguments
console.log(sum); // 8

Key terms:

  • function keyword → declares it
  • add → function name
  • (a, b) → parameters = placeholders for input
  • return → output value. If no return, function gives undefined
  • add(5, 3) → calling/invoking with arguments

3. Types of Functions

Function Declaration

Hoisted — you can call it before it’s defined.

sayHi(); // Works!

function sayHi() {
  console.log("Hi");
}

Function Expression

Stored in a variable. Not hoisted.

const sayBye = function() {
  console.log("Bye");
};

sayBye(); // Must call after definition

Arrow Function — ES6+ Short Syntax

const multiply = (x, y) => {
  return x * y;
};

// Implicit return if one-liner
const square = n => n * n;

console.log(square(4)); // 16

4. Parameters vs Arguments

function login(username, password) { // parameters
  // ...
}

login("alex", "1234"); // arguments

Default Parameters

function greet(name = "Guest") {
  return `Hello, ${name}`;
}

greet(); // "Hello, Guest"
greet("Sarah"); // "Hello, Sarah"

5. Why Use Functions?

  • DRY — Don’t Repeat Yourself. Reuse code.
  • Organization — Break big problems into small pieces
  • Testing — Test one function at a time
  • Readability — calculateTax() is clearer than 10 lines of math

6. Common Mistakes

  • Forgetting to call: greet does nothing. greet() runs it
  • Forgetting return: Function returns undefined by default
  • Wrong parameter order: add(3, 5) vs subtract(3, 5) gives different results

7. Real Examples

// Calculate area
const getArea = (width, height) => width * height;

// Check if user can vote
function canVote(age) {
  return age >= 18;
}

// Format currency
const toUSD = amount => `$${amount.toFixed(2)}`;

console.log(getArea(5, 10)); // 50
console.log(canVote(20)); // true
console.log(toUSD(19.9)); // $19.90

Mastery Checklist:

  • Define functions with declarations, expressions, and arrows
  • Pass arguments and use parameters
  • Return values from functions
  • Use default parameters
  • Understand when/why to use functions
